# 五子棋在线对战

## 快速开始

1. 访问游戏网址：
   https://bit.ly/play-gobang

2. 选择游戏模式：
   - 人机对战：直接和 AI 对战
   - 在线对战：和其他玩家对战

## 在线对战说明

1. 玩家 A：
   - 点击"在线对战"
   - 等待系统生成房间号
   - 将房间号发送给你的朋友

2. 玩家 B：
   - 点击"在线对战"
   - 输入朋友发来的房间号
   - 开始游戏！

## 游戏特性
- 无需下载安装
- 随时随地对战
- 跨平台支持（手机/电脑）
- 实时对战体验

## 注意事项
1. 确保网络连接稳定
2. 建议使用 Chrome 或 Edge 浏览器
3. 如果对战断开，可以使用相同房间号重新连接

## 遇到问题？
- 刷新页面重试
- 检查网络连接
- 更换浏览器尝试
